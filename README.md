# Dynamic Background (동적 배경 전환) — SillyTavern Extension

유저가 보낸 메시지든 AI(캐릭터)가 출력한 메시지든, 방금 새로 올라온 메시지 내용을 LLM에게 다시 보여주고,
**SillyTavern에 이미 등록되어 있는 배경 이미지들 중 가장 맥락에 어울리는 것을 AI가 직접 선택**하게
하는 방식입니다. 파일명을 별도로 입력하거나 목록을 관리할 필요가 없습니다.

- 배경 이미지는 SillyTavern에 원래 있는 **배경 관리 메뉴**(채팅 화면 상단의 배경/그림 아이콘, 또는
  사용자 설정의 "Backgrounds")에서 평소처럼 업로드하면 됩니다.
- 이 확장은 그 목록을 자동으로 불러와서(`POST /api/backgrounds/all`), 메시지가 올라올 때마다
  그중 어떤 게 지금 장면과 어울리는지 AI에게 물어보고, 어울리는 게 있으면 자동으로 배경을 바꿉니다.

예: "방 안에 들어서자..." 같은 메시지가 나오면, 이미 업로드해둔 `room.png`가
(정확히 "방"이란 단어가 파일명에 없어도) 문맥상 가장 어울린다고 판단되어 선택됩니다.

## 동작 방식

1. 확장이 켜지면 `POST /api/backgrounds/all`을 호출해 SillyTavern에 이미 등록된 배경 이미지 파일명
   목록을 가져옵니다. (설정 패널의 "배경 목록 새로고침" 버튼으로 언제든 다시 불러올 수 있습니다.)
2. `CHARACTER_MESSAGE_RENDERED`(AI 메시지 렌더링)와 `USER_MESSAGE_RENDERED`(유저 메시지 렌더링) 이벤트를
   모두 감지해서, 누가 보냈든 방금 새로 올라온 메시지를 기준으로 판단합니다.
3. 해당 메시지 텍스트와 1번에서 가져온 파일명 목록을 함께 SillyTavern의 `generateQuietPrompt()`로
   LLM에 전달합니다. (채팅 UI에는 보이지 않는 "조용한" 백그라운드 생성 요청입니다.)
4. Chat Completion API를 쓰는 경우 JSON Schema 구조화된 출력으로 목록 중 하나(또는 `NONE`)만
   정확히 고르도록 강제합니다. 구조화된 출력을 지원하지 않는 API(Text Completion 등)라면 일반 텍스트
   응답에서 파일명을 추출하는 방식으로 자동 폴백합니다.
5. 선택된 파일로 배경을 부드럽게 크로스페이드 전환합니다(`/backgrounds/파일명` 경로에서 이미지를 불러옵니다).
   `NONE`이 선택되면 설정된 기본 배경으로 바꾸거나, 기본 배경이 없으면 현재 배경을 유지합니다.
6. 채팅을 전환하면(`CHAT_CHANGED`) 그 채팅의 마지막 메시지를 기준으로 다시 계산합니다.

배경은 SillyTavern 자체 배경 시스템(#bg1 등)을 직접 건드리지 않고, 확장이 추가한 전체화면 레이어
(채팅 UI 뒤, `z-index: -5`)에 표시됩니다. 이미지 자체는 SillyTavern이 원래 서빙하는 경로를 그대로 사용합니다.

## ⚠️ 비용/속도 관련 중요 참고

"메시지마다 자동 실행" 옵션을 켜두면, 유저 메시지든 AI 응답이든 메시지 1개당 배경 판단용 LLM 호출이
1번 추가로 발생합니다. API 과금이 있는 서비스를 쓰신다면 추가 비용이 들고, 유저가 메시지를 보내는
순간에도 실제 응답 생성 요청과 별개로 API 호출이 하나 더 나가서 체감 속도에 영향을 줄 수 있습니다.
부담스럽다면 자동 실행을 끄고, 필요할 때 설정 패널의 "마지막 메시지로 지금 다시 계산" 버튼으로 수동 실행하세요.

## 설치 방법

1. 이 폴더(`st-dynamic-background`) 전체를 SillyTavern 서버의 아래 경로 중 하나에 복사하세요.
   - 사용자 전용: `data/<user-handle>/extensions/st-dynamic-background`
   - 전역(전체 사용자 공용): `public/scripts/extensions/third-party/st-dynamic-background`
2. SillyTavern을 새로고침하고, 확장(퍼즐 조각) 메뉴에서 "Dynamic Background"를 활성화하세요.
3. (아직 안 하셨다면) SillyTavern의 배경 관리 메뉴에서 배경 이미지를 몇 개 업로드하세요. 파일명을
   장면을 알아볼 수 있게 지어두면 AI가 더 잘 고릅니다 (예: `dark_forest_night.png`, `cozy_room.png`).

## 사용 방법

1. 확장 설정 패널에서 "동적 배경 전환" 드로어를 엽니다. 상태 문구에 현재 인식된 배경 이미지 개수와
   목록이 표시됩니다. 새로 이미지를 업로드했다면 "배경 목록 새로고침"을 눌러주세요.
2. 필요하면 "추가 지침"에 판단 기준을 자유롭게 적어줄 수 있습니다. (예: "애매하면 실내 배경 우선")
3. "매칭 실패(NONE) 시 기본 배경"에서 원하는 기본값을 고를 수 있습니다.
4. "메시지마다 자동 실행"을 켜두면 매 메시지마다 자동으로 배경이 바뀝니다. 끄면 수동 버튼으로만 동작합니다.
5. "마지막 메시지로 지금 다시 계산" 버튼으로 현재 상태를 바로 테스트해볼 수 있습니다.

## 문제 해결

- 설정 패널 상태 문구에 "등록된 배경 이미지가 없습니다"라고 뜬다면: SillyTavern 배경 관리 메뉴에서
  이미지를 아직 하나도 안 올린 상태입니다. 몇 개 업로드한 후 "배경 목록 새로고침"을 눌러주세요.
- 배경이 안 바뀐다면: 브라우저 개발자 도구(F12) → Console 탭에서 `[dynamic_background]`로 시작하는
  로그/에러가 있는지 확인해주세요. Text Completion API를 쓰는 경우 구조화된 출력이 안 먹혀서
  폴백 파싱이 실패했을 수 있습니다 — 이 경우 모델이 파일명을 텍스트로 정확히 뱉었는지 콘솔에서 확인하세요.
- "메시지마다 자동 실행"이 꺼져 있으면 아무 것도 자동으로 바뀌지 않습니다 (의도된 동작입니다).
