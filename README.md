# Dynamic Background (동적 배경 전환) — SillyTavern Extension

AI(캐릭터)가 출력한 메시지 텍스트를 감지해서, 미리 등록한 키워드가 포함되면
로컬 이미지 폴더의 배경으로 부드럽게 크로스페이드 전환하는 SillyTavern UI 확장입니다.

## 동작 방식

- `CHARACTER_MESSAGE_RENDERED` 이벤트(캐릭터 메시지가 화면에 렌더링된 직후)를 감지합니다.
- 해당 메시지 텍스트를 소문자로 변환 후, 설정에 등록된 규칙을 **위에서부터 순서대로** 검사합니다.
- 규칙의 키워드(쉼표로 여러 개 등록 가능) 중 하나라도 메시지에 포함되면, 그 규칙에 연결된 이미지로 배경을 전환합니다.
- 아무 규칙도 매칭되지 않으면 "기본 배경 이미지"로 전환하거나(설정된 경우), 현재 배경을 유지합니다.
- 채팅을 다른 캐릭터/대화로 전환하면(`CHAT_CHANGED`), 그 채팅의 마지막 AI 메시지를 기준으로 배경을 다시 맞춥니다.
- 배경은 SillyTavern 자체 배경 시스템을 건드리지 않고, 확장이 직접 추가한 전체화면 레이어(채팅 UI 뒤, `z-index: -5`)에 표시됩니다. 두 개의 레이어를 opacity로 교차 전환해서 자연스럽게 페이드됩니다.

## 설치 방법

1. 이 폴더(`st-dynamic-background`) 전체를 SillyTavern이 설치된 서버의 아래 경로 중 하나에 복사하세요.
   - 특정 사용자 전용: `data/<user-handle>/extensions/st-dynamic-background`
   - 전체 사용자 공용(전역 설치): `public/scripts/extensions/third-party/st-dynamic-background`
2. SillyTavern을 새로고침하고, 좌측 확장(퍼즐 조각) 메뉴에서 "Dynamic Background (동적 배경 전환)"이 보이는지 확인, 활성화하세요.
3. **중요**: `index.js` 상단의 `EXTENSION_URL_PATH` 값을 실제 설치 경로에 맞게 확인/수정하세요.
   - 전역(third-party) 설치라면 기본값(`scripts/extensions/third-party/st-dynamic-background`)을 그대로 쓰면 됩니다.
   - 사용자 전용 폴더에 설치했다면 브라우저 개발자도구(F12)의 네트워크 탭에서 확장 파일이 실제로 로드되는 URL 경로를 확인해서 그 값으로 바꿔주세요.
4. `backgrounds/` 폴더 안에 원하는 배경 이미지를 넣으세요 (jpg, png, webp 등 브라우저가 표시 가능한 형식이면 무엇이든 가능).

## 사용 방법

1. 확장 설정 패널(왼쪽 확장 메뉴)에서 "동적 배경 전환" 드로어를 엽니다.
2. 각 규칙 행에 "키워드"(쉼표로 여러 개 가능, 예: `숲, 나무, forest`)와 "이미지 파일명"(예: `forest.jpg`, `backgrounds/` 폴더에 실제로 존재해야 함)을 입력합니다.
3. "+ 규칙 추가" 버튼으로 규칙을 더 추가할 수 있고, ✕ 버튼으로 삭제할 수 있습니다.
4. "마지막 메시지로 테스트" 버튼을 누르면 현재 채팅의 마지막 AI 메시지 기준으로 즉시 배경 매칭을 테스트해볼 수 있습니다.
5. 전환 속도(ms)와 기본 배경(매칭 실패 시)도 설정할 수 있습니다.

기본값으로 숲/바다/밤/교실 관련 예시 규칙 4개가 미리 들어있습니다. 실제 사용 전에 `backgrounds/` 폴더에 해당 이미지가 있는지 확인하거나, 본인 시나리오에 맞게 규칙을 전부 수정/삭제하세요.

## 한계 및 참고사항

- 현재는 **키워드/문자열 포함 매칭** 방식입니다. AI 응답에 "숲"이라는 단어가 실제로 등장해야 매칭되며, 문맥을 이해하는 것은 아닙니다.
- 더 정교하게 하고 싶다면(예: AI가 명시적으로 장면을 언급하지 않아도 분위기를 유추), SillyTavern의 `generateQuietPrompt()`를 이용해 매 메시지마다 "이 장면에 어울리는 배경 태그 하나만 출력해줘" 같은 조용한 프롬프트를 추가로 호출해 분류하는 방식으로 확장할 수 있습니다. 다만 이 경우 메시지마다 추가 API 호출이 발생합니다.
- 설정은 SillyTavern의 `extensionSettings`에 저장되며, 서버의 사용자 설정 파일에 함께 저장됩니다.

## 파일 구성

```
st-dynamic-background/
├── manifest.json      # 확장 메타데이터
├── index.js           # 핵심 로직 (이벤트 감지, 배경 전환, 설정 UI)
├── style.css           # 배경 레이어 및 설정 패널 스타일
├── backgrounds/        # 배경 이미지를 직접 넣는 폴더
│   └── README.txt
└── README.md
```
